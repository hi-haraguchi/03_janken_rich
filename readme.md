# 力士ゲーム

## 制作サイトの説明

- 相撲にあまり触れたことがない人を対象に、相撲を身近に感じてもらうため
- ゲームをしながら力士の名前覚えるアプリをつくりました。
- 基本機能は以下の通りです。
  1. 「新しい力士」ボタンを押すとランダムに画像を表示
  2. 正解を押すと相手を少し押し出し、不正解だと少し自分が押し出される
  3. クイズをくり返して、どちらかが土俵の外に出ると、勝ち負け２種類の画像を表示

## 工夫した点・こだわった点

- 正解か不正解で左右にずれるだけでなく、ある確率で「投げ」で勝利できるようにしてあります。
- また左側にずれて自分が負けそうになると「投げ」の確率があがるようにしています。
- 対戦している力士の写真はグリッドレイアウトで配置しました。

## 難しかった点・次回トライしたこと

- より相撲に親しんでもらえるように前回の課題で作成したクイズにゲームをつなげようとしましたが、そもそも楽しんでもらえるようなゲームのアイデアを考えるのが難しかったです。
- また、親しんでもらうようなデザインを考えることにも苦労しました。土俵の画像は引き延ばして粗くなってしまい別の表現の仕方もあっただろうと思います。次はもっと細部のデザインの技術も習得したいです。

## 備考（感想、シェアしたいこと等なんでも）

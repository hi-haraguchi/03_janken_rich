# 力士ゲーム

## 制作サイトの説明

- 相撲を身近に感じてもらうためゲームをしながら力士の名前覚えるアプリをつくりました。
- クイズの正誤によって力士が左右に移動して、土俵の外に出ると勝敗が表示されます。

## 工夫した点・こだわった点

- 正解か不正解で左右にずれるだけでなく、ある確率で「投げ」で勝利できるようにしてあります。
- また左側にずれて自分が負けそうになると「投げ」の確率があがるようにしています。
- 対戦している力士の写真はグリッドレイアウトで配置しました。
- 右下に関連ページのボタンを設置して、右横からスライドしてくるようにしました。

## 難しかった点・次回トライしたこと

- より相撲に親しんでもらえるように前回の課題で作成したクイズにゲームをつなげようとしましたが、そもそも楽しんでもらえるようなゲームのアイデアを考えるのが難しかったです。
- また、デザインの作成にも苦労しました。土俵の画像は引き延ばして粗くなってしまい別の表現の仕方もあっただろうなと思います。次はもっと細部のデザインの技術も習得したいです。

## 備考（感想、シェアしたいこと等なんでも）

- JavaScript や jQuery でいろいろなことができることはよく分かりましたが、コードを書くにつれ、まだまだ勉強しないといけないことが多そうだなと感じています。
- 道のりは長そうですが、一歩ずつできることをコツコツ頑張っていきます。
